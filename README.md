# 🎮 Jogos JS Lab

Este repositório é um laboratório para testes e experimentos com criação de jogos usando JavaScript.

A ideia é explorar conceitos, testar mecânicas e aprender na prática como funciona o desenvolvimento de jogos.

## 🚀 Objetivo

- Experimentar ideias rapidamente  
- Testar bibliotecas e engines  
- Aprender conceitos de game dev na prática
