# 🎮 Dodge Game

O objetivo do jogo é desviar dos inimigos pelo maior tempo possível enquanto a pontuação aumenta continuamente.

Este projeto foi criado com foco em estudos práticos de desenvolvimento de jogos e exploração de funcionalidades importantes da engine.

---

# 🕹️ Como jogar

## Controles

- `A` → mover para esquerda
- `D` → mover para direita

## Objetivo

Desvie dos inimigos que caem do topo da tela e tente alcançar a maior pontuação possível.

Ao colidir com um inimigo, o jogo termina e a pontuação final é exibida.

---

# 📚 Conceitos estudados

Este projeto foi desenvolvido principalmente para praticar:

- Spawn dinâmico de inimigos
- Uso de múltiplas scenes
- Persistência de valores entre cenas
- Sistema de score
- Eventos de colisão
- Movimentação de objetos
- Loops temporizados (`loop`)
- Destruição dinâmica de objetos
- Uso de tags para sistemas automáticos
- Centralização e alinhamento com `anchor`

---

# ⚙️ Funcionalidades

- Tela inicial
- Tela de game over
- Sistema de score progressivo
- Spawn aleatório de inimigos
- Múltiplos tipos de inimigos
- Reinício da partida
- Cleanup automático de inimigos fora da tela

---

# 📌 Objetivo do projeto

O foco deste projeto foi aprender na prática como estruturar sistemas simples de gameplay, especialmente:

- criação automática de inimigos
- gerenciamento de cenas
- passagem de dados entre cenas
- controle de entidades via tags
